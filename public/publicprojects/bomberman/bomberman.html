Bomberman

a two player game


controls

player 1:

  • move left - left arrow
  • move up - up arrow
  • move right - right arrow
  • move down - down arrow
  • bomb - 0 key

player 2:

  • move left - a key
  • move up - w key
  • move right - d key
  • move down - s key
  • bomb - b key

Game Rules

Don't die

You can spawn more bombs at once by collecting green B bombs

Bomb range increases by collecting the + powerups

The game has 3 worlds on every start random one is chosen.

You can click here and create a new game world, copy the result code and create another world array. Paste in the generated values and add the name of new array to the worlds array in the bomberman.js file to run it
